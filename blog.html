Question-3: What is event delegation in the DOM

Event delegation is a way to add events in elements..it adds one event listener to it's parent element...then events capability reflects to the child elements it is effective and time saving approach it works futuristic added child elements too.

Question-4: What is event bubbles in the DOM

Event bubbling is when event happen to child element, but it go up to parent elements too...If child and parent both have event, first child event run, then parent event run. Event not stop at child, it keep going up. This useful for event delegation, because parent can handle many child events...If not want event go up, can use event.stopPropagation(). Event bubbling make code easy and no need to add event on every child
